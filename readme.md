# Heimdall

Plataforma CI/CD sencilla utilizando nodejs


## Requisitos

Runtime: NodeJS 10

Database: Mongo Atlas 


## Instalar dependencias

```
npm install
```

## Iniciar app

``` javascript
Core: 
npm start



View:
npm run view
```
